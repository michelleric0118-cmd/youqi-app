## 🎯 立即上线操作指南

### 方案A: GitHub Pages快速上线

#### 步骤1: 准备GitHub仓库
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/youqi-app.git
git push -u origin main
```

#### 步骤2: 部署
```bash
npm run deploy:github
```

### 方案B: Vercel快速上线
1. 访问 vercel.com
2. 使用GitHub账号登录
3. 导入GitHub仓库
4. 自动部署
