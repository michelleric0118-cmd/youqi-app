## 技术决策记录

### 1. 路由系统
- **选择**: React Router v6
- **原因**: 官方推荐，功能完整，社区支持好
- **状态**: ✅ 已集成

### 2. 组件架构
- **选择**: 函数式组件 + Hooks
- **原因**: 现代化，性能好，代码简洁
- **状态**: ✅ 已采用

### 3. 样式方案
- **选择**: CSS变量 + 工具类
- **原因**: 轻量级，易于维护，符合设计系统
- **状态**: ✅ 已实现

### 4. 状态管理
- **选择**: React Hooks (useState, useEffect)
- **原因**: 简单项目足够，避免过度工程化
- **状态**: ✅ 已采用

## 遇到的问题

- **问题**: 移动端摄像头权限问题
- **原因**: iOS/Safari需要HTTPS环境
- **解决方案**: 暂时跳过，后续用Vercel部署时解决
- **状态**: ⏸️ 已暂停
